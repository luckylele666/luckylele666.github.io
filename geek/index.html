数字移民ing

本文最后更新于：September 1, 2020 pm

选择方法论

虽然不是程序员，开发者但是个生活 geek。简体中文互联网世界环境恶劣，有监控审查，低劣广告，使用体验不佳等情况。真正有「数字移民」概念已经是 2019 年了，后知后觉。现今，要想注册一个网站或者 App 账号，无非只有用手机号注册和微信/微博等账号注册。他们可真笃定我们都是微信，微博的「忠实」用户呢，手机号更换了该怎么办呢？非常麻烦。前网络时代的通行证——邮箱，不得使用，反之国外很多网站和服务仍然可以使用邮箱注册，还能很方便的更换登录邮箱账号。

想要「数字移民」， 代理服务 的质量非常重要，如今的我不能忍受「断网」。

大部分应用都是即时更新，有的需仔细阅读更新描述再做决定。经验告诉我少部分东西基本上不要去更新，譬如：旧机型的 iOS 大版本；Windows 驱动，更新容易出问题；Bose 耳机固件，更新降噪能力削弱 (实锤)。

在此不介绍软件/服务的背景和使用方法，无特殊标注均为 iOS 和 Windows 系统环境。

江湖必备：

  • 给力的代理服务
  • Gmail (外网通行证)
  • Google Voice
  • 美区 Apple ID (支付绑定 PayPal )
  • 美区/港区 PayPal (少部分需要)

1. 上网神器🚀

  • Quantumult x
  • Surge
  • Shadowrockets

具体请看 神奇軟體。

2. Browser

御三家：

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Brave (期待发展)
  • Tor Browser (很少用)

鉴于 PC 端原本就用 Chrome, 使用 Google 全家桶方便，各种扩展，实验性功能好用。但是对隐私的看重，我选择迁移至 Firefox，数一数二的隐私安全保护，更好的字体渲染和个性化定制功能 (非常赞)。

3. 云服务☁

网盘之选 (免费容量)：

  1. iCloud (5GB)
  2. Google Drive (15GB)
  3. Microsoft OneDrive (5GB)
  4. Dropbox (2GB)
    • Google Photos (原图 15GB, 只用来放照片，后期考虑转移)

把 iPhone 的照片都转移到美区 iCloud (强烈推荐)和 Google photos。要想不充值快速从百度网盘/微云下载文件，尝试了油猴插件「网盘直链助手」+ IDM, 一次性只能下载几张照片且会报错，转而找到第二种方法——PanDownload, 只需要登录就可以快速多线程下载，免费使用，给个赞，虽然依此办法也花了数个小时来完成数据迁移。

4. 流媒体📺

  • Netflix (土耳其区付款，美区看)
  • Amazon Prime Video
  • HBO:《权力的游戏》、《西部世界》
  • Hulu (葫芦? )
  • Apple TV+ (真香)
  • Disney+ (手握无数 IP)
  • YouTube Premium
  • Fox+

从 B 站和国内御三家(BAT 旗下)的视频网站转向 YouTube (没有弹幕反而可以专心看内容)，画质好太多。作为果粉，当然选择 Apple TV+ ，有活动，一年免费看！墙内直连(偶尔有干扰)，根据网速自动选择画质 (最高 4K)，原声对白，中文字幕 (比起双语来居然不错)。苹果自制剧的水准也很 Apple~

5. Music🎵

  • Apple Music
  • Spotify (Boss)
  • Amazon Music

又是我大苹果耀武扬威的时候了~ 国区三个月 + 美区四个月的订阅体验。吐槽下界面略显简陋，没有听歌识曲功能。

6. Podcast

  • Apple Podcasts (祖师爷)
  • Pocket Casts (个人认为御三家之首)
  • Castro
  • Overcast
  • Agora (尝鲜)

Pocket Casts 界面好看 (颜值即正义)，全平台兼容，Apple Podcasts 被下架的内容都有，网页版需要付费。

7. 文件/笔记/Markdown📃

  • Evernote (原版印象笔记)
  • Notion (打算学习使用)
  • Typora (正在写此文)
  • Bear (iOS 备用)
  • Day One (老牌日记 App)

印象笔记迁出也非常麻烦，会员才可「一键迁移」，之前的订阅会员拜拜浪费…… 手动复制-粘贴好久 (doge)。

8. Password Manager

  • 1password
  • LastPass
  • KeePass (开源，本地，geek 之选)

活动免费的账户被 Frozen ，之后又放出家庭版一年订阅活动，再激活，不然我可就要唾弃了。导出密码非常麻烦！同账号移动密码找了半天，导出密码一定要在订阅结束前！

9. Sport🚲

  • Strava (Triathlon)
  • Garmin
  • Keep (累计 545 天，24649 分钟)

2016 年起已经在使用 Strava，花 $59.99 开了一年的 Summit (去年汇率低)。 iOS 上未被墙，Android 比较惨需要代理。这两个「小众」运动 App，都支持更换注册邮箱，赞！Garmin 在没有代理情况下会抽风。

10. RSS📓

  • Feedly
  • Inoreader
  • Reeder 4 (买断)
  • Unread 2 (订阅)
  • NetNewsWire 5.0 (免费)
  • Apple News (让自己多看英文)

Feedly 和 Inoreader 二虎相争，选择颜值高的，操作简单。Reeder 不用多说，Apple 生态最强。

11. 影音处理📸

  • VSCO (简简单单就是好用)
  • Snapseed
  • Vue (为数不多的国产精品)
  • iMovie (曾在 iOS 端剪了一个展示视频)
  • Clips (果家，备用)

VSCO 重新注册美区账号，简单。

12. To Do

  • Apple Reminders (iOS 🐂🍺)
  • TickTick (滴答清单)
  • Things 3 (iOS 付费王者，曾用过)
  • Microsoft To Do
  • Todoist
  • Do. List (还没删掉，可能有用吧)

用来用去，还是原生 Reminders 最好用…… 早日换本子和 macOS 联动。

13. Read it later🔖

  • Pocket
  • Instapaper

又是二选一，我选择颜值高的。全平台，开创 Read it later (从此吃灰), 被 Mozilla 收购，哪来的钱？！

14. Dictionary📚

  • Google Translate
  • Eudic (欧陆词典，付费)
  • Bing Dictionary

沙拉查词了解一下，几乎想要的都有。

15. Others⛏️

  • Speedtest (测速测速狂魔)

  • 熊猫吃短信 (也就那样啦)

  • ASUS Router (路由器の王者)

  • Fileget (iOS 下油管视频)

  • PIN (无敌剪贴板)

作为一名伪极简主义者没想到有这么多 App。工具虽好，但最厉害的还是自己的大脑，需要修炼「内核」后强力驱动。
